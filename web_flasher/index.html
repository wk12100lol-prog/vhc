⚡ Jak wgrać firmware na ESP8266?

1. Pobierz plik vhc_miner.bin z panelu obok.
2. Ściągnij program ESPHome-Flasher (Windows) lub esptool (Python).
3. Podłącz ESP8266 przez USB, wybierz port COM i plik .bin, kliknij Flash ESP.
4. Po zakończeniu podłącz ESP8266 do zasilacza USB – zacznie kopać automatycznie.

Wymagania: ESP8266 (NodeMCU v3), przewód microUSB, zasilacz 5V.
